## 🎯 设计原则（来自 website-ux-design skill）

### 1. 3 秒规则 ✅
用户 3 秒内理解：
- 这是什么网站 → 北京陪诊服务
- 提供什么价值 → 专业医疗协助
- 下一步做什么 → 免费咨询 CTA

### 2. 视觉层次 ✅
- **最重要:** Hero 标题 + CTA（最大、最亮）
- **次重要:** 服务卡片、价格（中等大小）
- **第三层:** 详细描述、FAQ（小字）

### 3. 一致性 ✅
- 颜色：主色紫色系，强调色粉色
- 字体：系统字体栈，统一字号比例
- 间距：8px 网格系统

### 4. 移动端优先 ✅
- 触摸目标 ≥ 44px
- 字体 ≥ 16px
- 无悬停依赖交互

## 🔧 技术细节

### 动画性能优化
- 使用 `transform` 和 `opacity`（GPU 加速）
- 避免 `left/top/width/height` 动画
- 减少动画元素数量

### 响应式断点
```css
/* 移动端 */
@media (max-width: 768px) { }

/* 平板 */
@media (min-width: 769px) and (max-width: 1024px) { }

/* 桌面 */
@media (min-width: 1025px) { }
```

### 浏览器兼容性
- ✅ Chrome/Edge (最新)
- ✅ Safari (最新)
- ✅ Firefox (最新)
- ⚠️ IE11 (不支持毛玻璃效果)
