## 🎯 关键成功因素

### 必须做到的：
1. ✅ **响应速度快** - 2 小时内回复
2. ✅ **专业度** - 熟悉医院流程、医学术语
3. ✅ **透明度** - 价格、流程、预期都说清楚
4. ✅ **可靠性** - 准时、守信、说到做到
5. ✅ **同理心** - 理解客户的焦虑和不适

### 绝对不能犯的错：
1. ❌ 迟到
2. ❌ 翻译错误（特别是医嘱）
3. ❌ 隐瞒费用
4. ❌ 服务后不跟进
5. ❌ 泄露客户隐私

## 🚨 紧急情况处理

### 客户突发急症
```
1. 立即拨打 120（急救电话）
2. 通知客户紧急联系人
3. 陪同前往最近医院急诊
4. 协助办理住院手续
5. 持续跟进直到家属到达
```

### 客户投诉
```
1. 立即道歉，不辩解
2. 了解具体问题
3. 提出解决方案（退款/重做/补偿）
4. 跟进直到客户满意
5. 记录教训，改进流程
```

### 医疗纠纷
```
1. 明确我们不是医疗服务提供方
2. 协助客户与医院沟通
3. 必要时提供翻译协助
4. 建议客户联系保险公司
5. 保留所有沟通记录
```
