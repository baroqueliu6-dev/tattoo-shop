## 🔧 需要配置的内容

### 1. PayPal Client ID（必须）
**位置:** `index.html` 底部
```html
<script src="https://www.paypal.com/sdk/js?client-id=YOUR_CLIENT_ID&currency=USD"></script>
```

**获取方法:**
1. 登录 PayPal Developer Dashboard
2. 创建 App
3. 复制 Client ID
4. 替换 `YOUR_CLIENT_ID`

### 2. YouTube 视频 ID（建议）
**位置:** `index.html` Video 板块
```html
<iframe src="https://www.youtube.com/embed/VIDEO_ID_1"></iframe>
```

**替换方法:**
- `VIDEO_ID_1` → 第一个视频的 YouTube ID
- `VIDEO_ID_2` → 第二个视频的 YouTube ID
- `VIDEO_ID_3` → 第三个视频的 YouTube ID

**YouTube ID 示例:**
- 视频 URL: `https://www.youtube.com/watch?v=dQw4w9WgXcQ`
- ID: `dQw4w9WgXcQ`

### 3. Formspree 表单 ID（必须）
**位置:** `index.html` Contact 表单
```html
<form action="https://formspree.io/f/YOUR_FORM_ID" method="POST">
```

**获取方法:**
1. 注册 Formspree (免费)
2. 创建新表单
3. 复制表单 ID
4. 替换 `YOUR_FORM_ID`

### 4. WhatsApp 号码（建议）
**位置:** `index.html` 多处
```
+86 138 0138 0000
```

**替换为你的真实号码**

### 5. Calendly 链接（建议）
**位置:** Hero 区域 "Book Now" 按钮
```html
<a href="https://calendly.com" target="_blank">
```

**替换为你的 Calendly 预约链接**
