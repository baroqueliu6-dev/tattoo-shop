## ✅ 修改详情

### 1. contact-form.html - 表单提交方式

**修改内容：**
- 将 Formspree 后端提交改为 mailto: 方案
- 添加 `sendEmail()` JavaScript 函数
- 用户点击提交后自动打开默认邮箱客户端

**修改位置：**
```html
<!-- 原代码 -->
<form id="contactForm" action="https://formspree.io/f/your-form-id" method="POST">

<!-- 修改后 -->
<form id="contactForm" onsubmit="sendEmail(); return false;">
```

**新增 JavaScript 函数：**
```javascript
function sendEmail() {
    const name = document.querySelector('[name="name"]').value;
    const email = document.querySelector('[name="email"]').value;
    const contact = document.querySelector('[name="contact"]').value;
    const message = document.querySelector('[name="message"]').value;
    
    const subject = `Contact Inquiry from ${name}`;
    const body = `Name: ${name}%0D%0AEmail: ${email}%0D%0AContact: ${contact}%0D%0A%0D%0AMessage:%0D%0A${message}`;
    
    window.location.href = `mailto:hello@13801380.site?subject=${subject}&body=${body}`;
}
```

**优点：**
- 无需后端服务
- 无需第三方依赖
- 自动打开用户默认邮箱
- 收件人、主题、正文已预填

### 2. patient-guide.html - 5 处内容优化

#### 修改 1: 新增中国免签政策解读

**位置：** Before You Come → Medical Visa Requirements 下方

**新增内容：**
- 72 小时过境免签政策（53 国公民）
- 144 小时过境免签政策（53 国公民）
- 海南 30 天免签政策（59 国公民）
- 申请要求和注意事项
- 黄色提示框引导用户联系确认资格

#### 修改 2: "We handle" → "We assist with"

**位置：** During Treatment → Hospital Registration Process

**原文案：**
> We handle check-in procedures and ensure all paperwork is completed correctly.

**修改后：**
> We assist with check-in procedures and ensure all paperwork is completed correctly.

**原因：** "handle" 暗示全权负责，"assist" 更准确（我们协助，客户自己决定）

#### 修改 3: "partner hospitals" → "recommended hospitals"

**位置：** Helpful Resources → Beijing Hospital Map

**原文案：**
> Interactive map showing all our partner hospitals with locations and specialties.

**修改后：**
> Interactive map showing recommended hospitals in Beijing with locations and specialties.

**原因：** 避免"合作"表述，改为"推荐"更准确

#### 修改 4: Medical Translation Glossary 说明

**位置：** Helpful Resources → Medical Translation Glossary

**原文案：**
> Common medical terms in English and Chinese for better communication.

**修改后：**
> Common medical terms in English and Chinese. Available after consultation.

**原因：** 明确这是付费咨询后提供的资源

#### 修改 5: 删除"免费"表述

**检查结果：** 
- 全文搜索"free"，仅发现"Visa-Free"（免签政策）表述，无需修改
- CTA 按钮已为"Get Assessment ($20)"，正确显示付费
