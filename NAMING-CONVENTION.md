## ✅ 检查清单

### 新增项目时
- [ ] 业务类型是什么？（产品=shop，服务=site）
- [ ] GitHub 仓库名是否统一？
- [ ] Vercel 项目名是否清晰？
- [ ] 本地目录结构是否一致？
- [ ] 域名配置是否正确？
- [ ] 是否更新 MEMORY.md 和 TOOLS.md？

### 操作前确认
- [ ] 仓库名从 GitHub 复制粘贴（不凭记忆）
- [ ] 本地路径用 `ls -la` 验证
- [ ] Git remote 配置正确（`git remote -v`）
- [ ] Vercel Root Directory 与实际路径匹配
