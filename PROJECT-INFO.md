## 🚀 部署流程

### 推送代码
```bash
cd ~/.openclaw/workspace/tattoo-shop
git add -A
git commit -m "描述更改"
git push origin main
```

### Vercel 自动部署
- 推送到 `main` 分支后自动部署
- 访问 Vercel Dashboard 检查部署状态
- 部署完成后域名自动更新

### 配置域名（如需要）
1. Vercel Dashboard → Settings → Domains
2. 点击 "Add Domain"
3. 输入域名（如 `13801380.site`）
4. 在域名注册商配置 DNS
5. 等待生效（通常 5-30 分钟）
