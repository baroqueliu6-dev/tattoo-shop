# Oriental Symbols - 产品素材

## 📦 文件说明

每个产品包含以下文件：

### 1. SVG 矢量文件
- `love-character-new.svg` - 爱 (Love)
- `joy-character.svg` - 乐 (Joy)
- `cool-character.svg` - 酷 (Cool)
- `good-character.svg` - 好 (Good)
- `beauty-character.svg` - 美 (Beauty)
- `history-character.svg` - 史 (History)

**用途：** 纹身师可以直接使用，无限缩放不失真

### 2. PNG 图片文件（用户已制作）
- `love-s.png` - 爱
- `happy-t.png` - 乐
- `cool-s.png` - 酷
- `good-s.png` - 好
- `beauty-s.png` - 美
- `history-s.png` - 史

### 2. PNG 图片文件（需要转换）
**如何生成 PNG：**

方法 1 - 使用在线工具：
1. 访问 https://cloudconvert.com/svg-to-png
2. 上传 SVG 文件
3. 设置分辨率为 300 DPI
4. 下载 PNG

方法 2 - 使用 macOS 预览：
1. 用「预览」打开 SVG 文件
2. 文件 → 导出
3. 格式选择 PNG
4. 分辨率设置为 300

方法 3 - 使用 Photopea（免费在线 PS）：
1. 访问 https://www.photopea.com
2. 打开 SVG 文件
3. 图像 → 图像大小 → 设置为 3000x3000px (300 DPI)
4. 文件 → 导出为 → PNG

### 3. 文化解读 PDF（需要转换）
- `love-meaning-new.html` → 爱
- `joy-meaning.html` → 乐
- `cool-meaning.html` → 酷
- `good-meaning.html` → 好
- `beauty-meaning.html` → 美
- `history-meaning.html` → 史

**如何生成 PDF：**
1. 用浏览器（Chrome/Safari）打开 HTML 文件
2. 按 Cmd+P (打印)
3. 目标打印机选择「另存为 PDF」
4. 保存

### 4. 产品封面图（需要制作）
建议尺寸：1280x720px

**设计建议：**
- 使用渐变背景（参考网站配色：#1a1a2e → #16213e）
- 中央放置大号汉字
- 底部添加产品名称和价格
- 保持简洁专业
